# Task System - Типы и логика

## Обзор

Типы описывают структуру данных для системы задач: категории, сами задачи и агрегированную статистику. Используются как единый контракт между компонентами и storage-логикой.
Все задачи поделены на типы, от типа зависит кол-во очков опыта, начисляемых за выполнение задачки.

## Основные типы

- small - мелкие задачи (5 очков)
- medium - средние задачи (10 очков)
- large - крупные задачи (50 очков)

### TaskCategory

```typescript
type TaskCategory = 'small' | 'medium' | 'large';
```
